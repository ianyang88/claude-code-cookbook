# Commands

这里收集 Claude Code 斜杠命令。

## 什么是 Command？

Command 是用户输入 `/xxx` 触发的预设 prompt，类似快捷方式。

## 如何使用

将命令文件放到你的项目 `.claude/commands/` 目录下。
