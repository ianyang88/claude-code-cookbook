# 核心命令详解

> 进入交互模式后（输入 `claude`），以下是日常高频操作

## 命令速查表

| 命令 | 功能描述 | 使用场景 |
|------|---------|---------|
| `/init` | 初始化项目 | 在新目录下运行，生成 `.claude.md` 上下文文件，定义项目规则 |
| `@ + 文件名` | 引用文件 | 输入 `@main.java` 或 `@src/`，将文件内容注入当前对话上下文 |
| `/model` | 切换模型 | 在 Sonnet 4.5, Opus 4.5 等模型间动态切换 |
| `/status` | 查看状态 | 检查 Token 使用情况、当前模型、上下文窗口剩余量 |
| `/compact` | 压缩上下文 | 自动摘要对话历史，剔除冗余试错过程，保留核心决策与 TODO |
| `Shift + Tab` | 切换运行模式 | 在编码模式（Accept Edits）与计划模式（Plan Mode）之间切换 |
| `/resume` | 恢复会话 | 当网络波动或意外退出时，找回之前的对话历史和上下文 |
| `/mcp` | 管理 MCP | 连接外部数据源或工具（如数据库、文件系统） |
| `/skills` | 加载技能包 | 调用预设的复杂任务模板（如"重构代码"、"编写单元测试"） |
| `/memory` | 管理长期记忆 | 查看或清除 AI 记住的项目特定偏好和历史信息 |

## 运行模式切换

**Shift + Tab** 可在两种模式间无缝切换：

| 模式 | 说明 |
|------|------|
| 编码模式 (Accept Edits) | 直接执行/写代码 |
| 计划模式 (Plan Mode) | 输出方案/步骤分析 |

控制 AI 是先思考还是先行动

## 模型选择建议

| 模型 | 适用场景 |
|------|---------|
| Sonnet 4.5 | 日常开发，平衡速度与智能 |
| Opus 4.5 | 复杂任务，需要最强推理能力 |
