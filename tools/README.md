# Tools 外部工具

增强 Claude Code 体验的第三方工具。

## 已收集的工具

| Tool | 类型 | 一句话介绍 | 适用场景 |
|------|------|-----------|---------|
| [ccstatusline](./ccstatusline.md) | 状态栏 | 高级状态栏 | 显示 Token/花费/上下文 |
| [tweakcc](./tweakcc.md) | 定制化 | 美化定制 | 自定义主题/动画 |
| [cc-connect](./cc-connect.md) | 远程控制 | 手机远程操作 | 飞书/钉钉/微信/Telegram |
| [claudecodeui](./claudecodeui.md) | 可视化 | Web 界面 | 桌面/移动端图形界面 |

## 工具分类

### 远程访问
- **cc-connect** — 通过飞书/钉钉/微信/Telegram 远程控制
- **claudecodeui** — Web 界面

### 视觉增强
- **tweakcc** — 主题、字体、动画定制
- **ccstatusline** — 状态栏美化
