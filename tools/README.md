# 工具推荐

这里收集 Claude Code 相关的第三方工具和增强工具。

## 已收集的工具

| 工具 | 类型 | 说明 |
|------|------|------|
| [cc-connect 远程控制](./cc-connect-remote-control.md) | 远程控制 | 手机远程控制 Claude Code，| [claudecodeui Web 界面](./claudecodeui-web-interface.md) | 可视化 | Web/移动端图形界面 |
| [tweakcc 美化定制](./tweakcc-customization.md) | 定制化 | 自定义主题、系统提示、思考动画 |
| [ccstatusline 状态栏](./ccstatusline-usage.md) | 状态栏 | 高级状态栏，显示模型/Git/Token 等指标 |

## 工具分类

### 远程访问
- **cc-connect** — 通过飞书/钉钉/微信/Telegram 远程控制
- **claudecodeui** — Web 界面，- **IDE 集成** — VS Code / IntelliJ 插件

### 视觉增强
- **tweakcc** — 主题、字体、动画定制
- **ccstatusline** — 状态栏美化
