# Skills

这里收集可复用的 Claude Code Skills 配置。

## 什么是 Skill？

Skill 是一个详细的 Markdown 文档，告诉 Claude 如何完成某类任务。它包含最佳实践、模式和注意事项。

## 如何使用

将 Skill 文件放到你的项目 `.claude/skills/` 目录下，Claude Code 会自动加载。
