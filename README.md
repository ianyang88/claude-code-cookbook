# Claude Code Cookbook

> 一句话优化你的 Claude Code — AI 全程引导

## 是什么

这是一个面向新用户的 Claude Code 优化向导。你只需说一句话，AI 自动了解你的需求，按优先级帮你安装配置。

## 怎么用

打开 Claude Code，说：

```
参考 https://github.com/ianyang88/claude-code-cookbook 帮我优化配置
```

AI 会问你几个问题（系统、用途、经验），然后按优先级帮你安装：
1. **必做**（5分钟）— 状态栏 + 黄金工作流
2. **推荐**（15分钟）— 主题定制 + 开发技能包
3. **按需**（按场景）— 远程控制 / 文档处理 / Web 界面

也可以直接看 [快速上手指南](./getting-started.md) 自行配置。

## 核心理念

不要把 Claude Code 当聊天机器人，它是**代理式编码者**。

| 心态 | 价值发挥 |
|------|----------|
| 当成 Prompt 工具 | 仅发挥 10% |
| 当成代理式编码者 | 发挥 100% |

**黄金工作流**：探索 → 规划 → 执行，不要上来就写代码。
