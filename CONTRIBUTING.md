# 贡献指南

## 欢迎贡献！

感谢你愿意分享 Claude Code 的使用经验。

## 如何贡献

### 贡献 Skills / Commands

1. Fork 本仓库
2. 在对应目录创建 `.md` 文件
3. 按照模板格式编写
4. 提交 PR

### 贡献 Guides

分享你的教程、案例或使用技巧。

### 贡献 Resources

推荐好用的工具或项目，编辑 [resources.md](./resources.md)。

## 文件命名规范

- 使用小写字母和连字符：`python-testing.md`
- 语义清晰：`git-workflow.md` 而非 `git.md`

## 内容要求

- 中文撰写（翻译内容请注明来源）
- 提供可运行的代码示例
- 描述清晰易懂

## 许可

提交内容即表示同意以 MIT 协议开源。
